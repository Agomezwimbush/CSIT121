XML and JavaScript

What is XML?

XML is the acronym for eXtensible Markup Language. XML is used for documents conatining structured information.

XML can transport platform information to another while ensuring clarity.

XML is just information wrapped in tags

XML is a markup language that defines a set of rules for encoding documents in a format that is both human-readable and machine-readable.

XML:

  • Is easier to read.
  • Is platorm-inependent.
  • Is just another piece of software that was created to send, receive, store or display data.
  • Just stores the data and doesn't do anything else with it.

What are some rules of XML?

  1. XML documents MUST have a root element and child elements or sub-elements.
  2. Opening and closing tags are case sensitive.
  3. Opening and closing tags must be properly nested.
  4. Whitespace characters between XML elements/attributes will be ignored.
  5. XML tags must be closed in the same order they are opened.

What is JavaScript?

JavaScript is a computer programming language that is used to make websites and applications more dynamic and interactive.

98% of all websites today use JavaScript.

An important part of JavaScript is working with the Document Object Model or DOM.

JavaScript Events:

  • click
  • load
  • mouseover

JavaScript Methods:

  • parseInt()
  • replace()
  • toUpperCase()

JavaScript Properties:

  • innerHTML
  • style
  • value

Functions in Javascript are the main parts of the program. They allowed the code to be called multiple times without repetition.

How is JavaScript related to HTML and CSS?

HTML is the structure of a webpage.

CSS is the style of a webpage.

JavaScript is the behavior of a webpage.
